# 示例：如何在文章中使用图片

## 方式 1：使用 assets 目录

```markdown
![示例图片](/assets/images/screenshot.png)
```

## 方式 2：使用外部图床

```markdown
![示例图片](https://img.shields.io/badge/example-image)
```

## 方式 3：使用占位图服务

```markdown
![示例图片](https://via.placeholder.com/800x400)
```

## 推荐做法

1. **大文件** - 使用图床或 CDN
2. **小图标** - 可以放在 `assets/images/`
3. **截图** - 考虑使用图床自动上传
4. **示例图片** - 使用占位图服务
