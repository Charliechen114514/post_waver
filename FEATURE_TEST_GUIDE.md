# 注入模板和相关链接功能测试指南

## 🎉 新功能概述

本次更新添加了两个主要功能：

1. **注入模板管理** - 创建和管理可复用的文章标题后注入内容
2. **相关链接显示** - 在文章预览中自动显示相关阅读链接

## 🚀 访问应用

开发服务器已启动：
- **Web UI**: http://localhost:5173/post_waver/
- **API Server**: http://localhost:3001

## 📝 测试步骤

### 1. 创建注入模板

1. 访问 http://localhost:5173/post_waver/templates
2. 点击"➕ 新建模板"按钮
3. 填写模板信息：
   - **名称**: 例如 "订阅引导"
   - **描述**: 例如 "在文章开头引导读者订阅"
   - **内容**: 例如 "🔥 欢迎订阅我的专栏，获取更多优质内容！"
   - **启用**: 勾选
4. 查看平台预览（掘金/微信/HTML）
5. 点击"保存"

### 2. 使用注入模板

1. 访问 http://localhost:5173/post_waver/publish（发布工作台）
2. 点击"🔄 扫描文章"获取文章列表
3. 选择一篇文章进行预览
4. 在"注入内容"下拉框中选择刚才创建的模板
5. 观察：预览内容应该在标题后显示注入的内容

### 3. 测试相关链接

1. 在发布工作台中，确保"显示相关链接"复选框已勾选
2. 选择一篇文章进行预览
3. 观察：预览内容末尾应该显示：
   - **相邻文章**: 上一篇/下一篇（如果存在）
   - **推荐阅读**: 相关文章及其相似度

### 4. 组合测试

1. 同时启用注入模板和相关链接
2. 检查预览效果
3. 尝试不同的平台（掘金、微信、HTML）
4. 验证内容格式是否正确

### 5. 管理模板

1. 回到模板管理页面
2. 测试编辑功能
3. 测试禁用/启用功能
4. 测试删除功能（确认是否需要）

## 🐛 可能的问题和解决方案

### 问题1: 看不到相关链接
**原因**: 文章还没有生成内容索引
**解决**: 运行 `pnpm workflow:scan` 扫描文章并生成索引

### 问题2: 模板选择器为空
**原因**: 还没有创建模板或所有模板都被禁用
**解决**: 访问模板管理页面创建或启用模板

### 问题3: 预览内容没有更新
**原因**: API服务器可能需要重启
**解决**: 重启开发服务器 `pnpm dev`

## 📊 技术细节

### 数据库变更
- 新增 `InjectionTemplate` 表用于存储模板
- 复用 `ContentIndex` 表获取相关链接信息

### API端点
- `GET /api/injection-templates` - 获取所有模板
- `POST /api/injection-templates` - 创建模板
- `PUT /api/injection-templates/:id` - 更新模板
- `DELETE /api/injection-templates/:id` - 删除模板
- `POST /api/posts/:id/preview` - 支持新参数：
  - `injectionTemplateId` - 应用注入模板
  - `includeRelatedLinks` - 是否包含相关链接

### 新增页面
- `/templates` - 模板管理页面
