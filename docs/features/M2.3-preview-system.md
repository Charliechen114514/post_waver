## 🎯 实施目标

实现一个轻量级的预览服务器，允许用户在浏览器中预览转换后的Markdown内容，并提供一键剪贴板复制功能，改善发布体验。

## ✅ 交付物

### 1. 预览服务器
**文件**: [`packages/engine/src/preview/server.ts`](../../packages/engine/src/preview/server.ts)

- 基于 Hono.js 的轻量级Web服务器
- 端口自动发现机制（3737+）
- 支持的健康检查端点：`/health`
- 预览端点：`/preview/:platform/:id`
- CORS支持
- 优雅的错误处理

**核心特性**:
```typescript
- 自动端口发现（避免冲突）
- URL编码支持（处理文件路径中的斜杠）
- 平台参数验证
- 友好的错误页面
```

### 2. 预览编排器
**文件**: [`packages/engine/src/preview/orchestrator.ts`](../../packages/engine/src/preview/orchestrator.ts)

- 协调文章解析、内容转换和服务器启动
- 集成 `@content-hub/core` 的 `parsePost()`
- 集成 `@content-hub/transformer` 的转换函数
- 支持的平台：`wechat`, `juejin`, `html`

**关键函数**:
```typescript
startPreview(postPath: string, platform: string): Promise<PreviewServer>
getPreviewURL(port: number, platform: string, id: string): string
```

### 3. 模板渲染器
**文件**: [`packages/engine/src/preview/templates.ts`](../../packages/engine/src/preview/templates.ts)

- 支持 `PreviewContext` 接口
- HTML转义处理
- 模板变量插值
- 为未来文件模板预留扩展性

### 4. 剪贴板集成
**文件**: [`packages/engine/src/clipboard.ts`](../../packages/engine/src/clipboard.ts)

- 使用 `clipboardy` 实现跨平台剪贴板操作
- 异步API设计
- 错误处理和用户反馈

**API**:
```typescript
copyToClipboard(content: string): Promise<void>
readFromClipboard(): Promise<string>
```

### 5. 策略调度器
**文件**: [`packages/engine/src/scheduler.ts`](../../packages/engine/src/scheduler.ts)

- 支持三种发布策略：`auto`, `semi-auto`, `copy`
- 统一的错误处理
- 可扩展的策略模式

**策略说明**:
- `auto`: 自动发布（未来功能）
- `semi-auto`: 打开预览页面，手动复制
- `copy`: 直接复制到剪贴板

### 6. CLI命令
**文件**: [`scripts/preview.ts`](../../scripts/preview.ts)

- 基于Commander.js的CLI工具
- 自动打开浏览器
- 优雅的关闭处理（Ctrl+C）
- 友好的用户提示

**新增命令**:
```bash
pnpm preview:wechat <file>  # 微信预览
pnpm preview:juejin <file>  # 掘金预览
pnpm preview:html <file>    # HTML预览
```

## 🏗️ 架构设计

### 数据流

```
CLI Command → Orchestrator → Preview Server → Browser
                ↓                 ↓
           parsePost()      Hono.js Routes
                ↓                 ↓
         transformers()   Template Engine
```

### 技术栈

| 组件 | 技术 | 用途 |
|------|------|------|
| 服务器 | Hono.js | 轻量级Web框架 |
| 服务器适配器 | @hono/node-server | Node.js HTTP服务器 |
| 剪贴板 | clipboardy | 跨平台剪贴板操作 |
| 浏览器打开 | open | 跨平台浏览器启动 |
| CLI框架 | Commander.js | 命令行界面 |

## 📝 使用示例

### 基本使用

```bash
# 预览微信格式
pnpm preview:wechat content/posts/my-article.md

# 预览掘金格式
pnpm preview:juejin content/posts/my-article.md

# 预览HTML格式
pnpm preview:html content/posts/my-article.md
```

### 高级用法

```typescript
import { startPreview, executeStrategy } from '@content-hub/engine'

// 启动预览
const server = await startPreview('content/posts/test.md', 'wechat')
console.log(`预览地址: http://localhost:${server.port}`)

// 使用策略调度器
const result = await executeStrategy(
  'content/posts/test.md',
  'wechat',
  'semi-auto'
)
```
