# M2.3 变更日志

## 版本信息

- **版本**: v0.2.3

- **状态**: ✅ 已完成

---

## 📦 新增功能

### 预览系统
- ✨ 新增轻量级预览服务器（基于Hono.js）
- ✨ 支持三个平台的预览：微信、掘金、HTML
- ✨ 一键剪贴板复制功能
- ✨ 自动打开浏览器预览
- ✨ 端口自动发现机制（3737+）

### CLI命令
- ✨ `pnpm preview:wechat <file>` - 微信预览
- ✨ `pnpm preview:juejin <file>` - 掘金预览
- ✨ `pnpm preview:html <file>` - HTML预览

### 策略调度器
- ✨ 支持三种发布策略：auto、semi-auto、copy
- ✨ 统一的错误处理接口
- ✨ 可扩展的策略模式
